# The Bounded Observer

A self-contained interactive plus walkthrough: how a bounded observer predicts a
computationally irreducible world, how its grain changes what is knowable, and how
coarse-graining produces the arrow of time. Part of the project "The Mirror and the Loom."

- `index.html` — the article with the live sandbox embedded (the GitHub Pages home).
- `sandbox.html` — the interactive only, sized for embedding in an iframe.

All client-side JavaScript. No build step, no dependencies, no network calls.
